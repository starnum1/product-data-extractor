# 商品数据提取器 Chrome 插件

一键提取电商页面商品的长宽高和重量数据。

## 功能特点

- 🚀 一键提取商品尺寸（长x宽x高）
- ⚖️ 自动识别重量数据
- 📋 一键复制所有数据
- 🌍 支持多语言（中文、俄语、英语）
- 🎯 智能识别数据卡片

## 安装方法

### 开发者模式安装

1. 打开 Chrome 浏览器
2. 访问 `chrome://extensions/`
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择 `product-data-extractor` 文件夹
6. 安装完成！

## 使用方法

1. 打开包含商品数据的页面（如 OZON、淘宝、京东等）
2. 等待数据卡片加载完成
3. 点击浏览器工具栏的插件图标
4. 点击"提取数据"按钮
5. 查看提取的尺寸和重量信息
6. 可以一键复制所有数据

## 支持的数据格式

### 尺寸格式
- `480 x 380 x 180mm`
- `480x380x180mm`
- `48 × 38 × 18 cm`

### 重量格式
- `7650g`
- `7.65kg`
- `7650 g`

### 支持的关键词
- 中文：长 宽 高、长宽高、尺寸、重 量、重量
- 俄语：размер、вес
- 英语：dimensions、size、weight

## 技术实现

- **Manifest V3**: 使用最新的 Chrome 扩展 API
- **Content Script**: 注入页面提取 DOM 数据
- **正则表达式**: 智能匹配各种数据格式
- **MutationObserver**: 监听动态加载的内容

## 注意事项

1. 插件需要在数据卡片加载完成后才能提取数据
2. 如果提取失败，请刷新页面后重试
3. 支持所有网站，但数据格式需要符合常见规范
